# Security Policy

## Reporting a vulnerability

Report security issues by emailing `security@konto.dev` or by opening a private vulnerability report through GitHub security advisories for this repository. We will acknowledge new reports and provide an initial response within 72 hours.

## What counts as a security issue

Any issue that can affect financial correctness, ledger integrity, transaction ordering, balance derivation, idempotency guarantees, data integrity, or the safety of funds counts as a security issue.
