# Introduction

Konto is a brutally fast, zero-sum enforced, double-entry ledger engine built on PostgreSQL.

## Features

- **Mathematical Integrity**: Triggers enforce `SUM(amount) = 0` prior to commit.
- **Pessimistic Locking**: Eliminates deadlocks via deterministic lexicographical sorting.
- **Fail-Fast**: Violently crashes on schema validation errors.
