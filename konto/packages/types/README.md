# @konto-ledger/types

Shared TypeScript interfaces for Konto database executors.

This package contains the core types and interfaces required by `@konto-ledger/core` and `@konto-ledger/adapters`, such as the `KontoQueryExecutor` interface that enables dependency injection for different database drivers.

## Installation

```bash
npm install @konto-ledger/types
```

For the full documentation and setup guide, please visit the main repository:
